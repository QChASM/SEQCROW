Anatomy of an ORCA Input File

You can find the full specification of ORCA input files in the ORCA manual, which can be downloaded from their online forum. The input builder splits this into sections:

  • Simple input - method, basis set, job type, and other options
  • Molecule - charge, multiplicity, atoms and their coordinates
  • Various blocks - some job types may require more input than what is in the simple input

General options for simple input, comments, and blocks can be modified on the "additional options" page. More commonly-used options have a dedicated widget or page on the tool window (e.g. basis sets).
