Anatomy of a Psi4 Input File

You can find the full specification of Psi4 input files in the Psi4 online documentation. The input builder splits this into sections:

  • Settings - global settings
  • Job - job type and computational method
  • Basis - basis set
  • Molecule - charge, multiplicity, atoms and their coordinates
  • Before job - miscellaneous lines that will appear before the job
  • After job - miscellaneous lines that will appear after the job
  • OptKing - settings for Psi4's geometry optimizer
  • Before molecule - miscellaneous lines that will appear before the molecule (and settings)
  • PCM solver - settings for an interface to PCMSolver, a library for computations using the polarizable continuum model for solvents
  • Comments - optional description of the job

General options for these (except basis) can be modified on the "additional options" page. Commonly-used options have a dedicated widget or page on the tool window (e.g. basis sets).
