Anatomy of a Q-Chem Input File

You can find the full specification of Q-Chem input files in the Q-Chem manual, which can be found online. The input builder splits the input file into different sections:

  • Comment - optional description of the job
  • Settings - the $REM section
  • Sections - all other sections aside from $REM
  • Molecule - atoms and their coordinates

General $REM settings, other sections, and and comments can be added on the "additional options" page. Commonly-used options have a dedicated widget or page on the tool window (e.g. methods). 

$rem
    MAKE_CUBE_FILES      =   true
    PLOTS                =   true
    BASIS                =   def2-SVP
    METHOD               =   B3LYP
$endSettings
$plots
    grid_range                     (-8,8) (-8,8) (-8,8)
    grid_points                    40 40 40
    total_density                  0-2
    transition_density             1-2
    attachment_detachment_density  1-2
    alpha_molecular_orbital        28-31
$end"plots" section
$comments
    test.out
$endcomments
