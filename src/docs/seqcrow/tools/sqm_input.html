Anatomy of an SQM Input File

You can find the full specification of Psi4 input files in the Amber manual, which can be downloaded from the Amber website. The input builder splits this into three sections:

  • Title - optional description of the job
  • QMMM namelist - everything except atoms and their coordinates
  • Molecule - atoms and their coordinates

General QMMM namelist options and comments can be added on the "additional options" page. Commonly-used options have a dedicated widget or page on the tool window (e.g. methods).

Note that if you are editing these files by hand, spaces at the start of each line and empty lines can be important.

    qmcharge=0,
    spin=1,
    maxcyc=0,
    qm_theory='AM1',
