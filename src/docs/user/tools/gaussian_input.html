Anatomy of a Gaussian Input File

You can find the full specification of Gaussian input files in the Gaussian manual, either online or your physical copy. The input builder splits this into sections:

  • Link 0 commands - control job resources and caching data
  • Route - method, basis set, job type, and other options
  • Comments (title) - may or may not be present; intended to be a description of the computation
  • Molecule - charge, multiplicity, atoms and their coordinates
  • Miscellaneous input - some job types may require more input than what is in the route, particularly when Gaussian interfaces with other programs (e.g. NBO) - these appear at the end of the file

Note that if you are editing Gaussian input files manually, the placement of blank lines can matter. See the Gaussian manual for details.

General options for link 0 commands, the route, comments, and miscellaneous input can be modified on the "additional options" page. More commonly-used options have a dedicated widget or page on the tool window (e.g. basis sets).
